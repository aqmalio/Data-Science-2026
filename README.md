## Tentang

Repositori ini saya buat sebagai perjalanan saya atau portofolio pembelajaran mendalami *Data Science*. Tujuan saya belajar *Data Science* adalah untuk memahami bagaimana cara mengekstrak informasi dari data mentah dan membangun model prediktif, yang ke depannya sangat relevan buat diintegrasikan dengan proyek-proyek *web development* atau *SaaS* saya.

Mulai dari manipulasi tabel data, eksplorasi data secara visual (EDA), hingga membangun model AI sederhana untuk memprediksi angka (Regresi Linear) dan melakukan klasifikasi (Regresi Logistik).

## Tools & Library

*   **Python 3:** Bahasa pemrograman.
*   **Pandas:** Manipulasi dan analisis data *tabular* (Dataframe).
*   **NumPy:** Untuk komputasi numerik dan operasi *array*.
*   **SciPy:** Digunakan untuk komputasi saintifik dan perhitungan statistik teknis yang lebih kompleks.
*   **scikit-learn:** *Library* utama untuk memanggil algoritma *Machine Learning* (Regresi Linear, Logistik, *scaling*, dll).
*   **Matplotlib & Seaborn:** Untuk visualisasi data, bikin grafik, dan nge-plot matriks.

#### 2. Menjalankan Secara Lokal di PC

1. Clone repository ini:
```bash
   git clone https://github.com/aqmalio/Data-Science-2026.git
```

2. Masuk ke direktori repository:
```bash
  cd Data-Science-2026
```

3. Buat dan aktifkan Virtual Environment
```bash
  # Membuat virtual environment bernama 'env'
   python -m venv env

   # Mengaktifkan virtual env di Linux/Mac:
   source env/bin/activate  
   
   # Mengaktifkan virtual env di Windows:
   env\Scripts\activate
```

4. Install semua library yang dibutuhkan:
```bash
  pip install pandas numpy scipy scikit-learn matplotlib seaborn jupyter
```

5. jalankan jupyter notebook
```bash
  jupyter notebook
```
