## Tentang

Repositori ini saya buat sebagai perjalanan saya atau portofolio pembelajaran mendalami *Data Science*. Tujuan saya belajar *Data Science* adalah untuk memahami bagaimana cara mengekstrak informasi dari data mentah dan membangun model prediktif, yang ke depannya sangat relevan buat diintegrasikan dengan proyek-proyek *web development* atau *SaaS* saya. 

Di dalam *repo* ini, terdapat kumpulan file Jupyter Notebook (`.ipynb`) yang mendokumentasikan progres belajar dari nol (fundamental) hingga implementasi *Machine Learning*. Mulai dari manipulasi tabel data, eksplorasi data secara visual (EDA), hingga membangun model AI sederhana untuk memprediksi angka (Regresi Linear) dan melakukan klasifikasi (Regresi Logistik).
